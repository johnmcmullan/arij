## Description

The app currently supports creating and viewing tickets, plus drag-and-drop status changes on the board. Missing: editing ticket fields after creation, deleting tickets, and updating status from the ticket detail page.

## Acceptance Criteria

- [ ] Tickets can be edited (title, description, assignee, status)
- [ ] Tickets can be deleted with confirmation
- [ ] Status can be changed from the ticket detail page
- [ ] Projects can be edited and archived
