## Description

Projects can be created and viewed but not edited or archived. Add `GET /projects/:id/edit`, `PUT /projects/:id`, and `PATCH /projects/:id/archive`.

## Acceptance Criteria

- [ ] Edit button on project detail page
- [ ] Edit form for name, key, and description
- [ ] Archive button sets `archived = 1` (project disappears from list)
- [ ] Archived projects still accessible via direct URL
