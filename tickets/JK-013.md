## Description

Add a labels system with a many-to-many relationship. Labels are freeform text tags (no predefined set). Support adding/removing labels on tickets and filtering by label.

## Acceptance Criteria

- [ ] `labels` table and `ticket_labels` junction table
- [ ] Comma-separated label input on ticket create/edit forms
- [ ] Labels displayed as colored chips on ticket detail and board cards
- [ ] Clickable labels filter the project view to matching tickets
